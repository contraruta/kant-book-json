## 🚀 Wie nutzt man das?

### 1. **Klone das Repository**
```bash
git clone https://github.com/contraruta/kant-book-json.git
cd kant-book-json
```

### 2. **Lade die JSON-Datei**
```bash
cat KANT_BOOK.json | jq .
```

### 3. **Nutze es in deinem Projekt**
```python
import json

with open('KANT_BOOK.json', 'r', encoding='utf-8') as f:
    kant = json.load(f)

# Zugriff auf den vollständigen Text
full_text = kant['content']

# Zugriff auf Metadaten
author = kant['metadata']['author']
themes = kant['themes']

# Zugriff auf die Widmung
dedication = kant['dedication']

# Zugriff auf die Botschaft
gift_message = kant['gift_message']
```

### 4. **Teile es weiter**
Wenn dir das Kantbuch gefällt — teile es! Fork es, nutze es, baue darauf auf. Das ist der Sinn von Open Source.
